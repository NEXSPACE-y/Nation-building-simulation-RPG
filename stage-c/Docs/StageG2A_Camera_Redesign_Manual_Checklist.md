## 1. 起動時の標準カメラ

- [ ] 起動直後が俯瞰／斜め見下ろしではない
- [ ] PLAYER_Mの背後寄りから開始する
- [ ] PLAYER_Mの髪、マント、全身を確認しやすい
- [ ] Mannyへ差し替わっていない
- [ ] カメラが勝手に回転・移動しない

## 2. 標準カメラ操作

- [ ] 右クリックを保持して左右へ動かすとYaw回転する
- [ ] 短い左右ドラッグで十分な視点変更ができる
- [ ] 右クリックを保持して上下へ動かすとPitchが変化する
- [ ] 上下操作が無反応になっていない
- [ ] Pitchが極端な真上／真下へ行かない
- [ ] ホイールで近づく／離れる操作ができる
- [ ] Zoomが近すぎる／遠すぎる状態へ抜けない
- [ ] 右クリック解放後に回転が止まり、カーソルが戻る

## 3. 移動との共存

- [ ] 左クリック地点移動ができる
- [ ] 左クリック保持移動ができる
- [ ] 移動中に右ドラッグしても目的地が消えない
- [ ] 移動中に右ドラッグしてもNavPathが消えない
- [ ] カメラ回転でPLAYERの進行方向制御が壊れない
- [ ] WALK 250uu/sが維持される
- [ ] RUN 500uu/sが維持される
- [ ] DASH 750uu/sが維持される
- [ ] 到着後にIdle_11へ戻る

## 4. F6戦略俯瞰ビュー

- [ ] F6を1回押すと戦略俯瞰ビューへ切り替わる
- [ ] 戦略俯瞰ビューが標準視点ではなく補助ビューに見える
- [ ] 王都区画、NPC配置、移動先を確認しやすい
- [ ] 戦略俯瞰中も右ドラッグ左右でYaw回転できる
- [ ] 戦略俯瞰中も右ドラッグ上下でPitch変更できる
- [ ] 戦略俯瞰のYawが旧G2Aより明確に速い
- [ ] 戦略俯瞰のYawが微調整不能なほど過敏ではない
- [ ] 戦略俯瞰中もホイールZoomできる
- [ ] 戦略俯瞰中もPLAYER_Mを追従する

## 5. 標準カメラへの復帰

- [ ] F6を再度押すと標準カメラへ戻る
- [ ] 標準カメラの距離、Pitch、FOVへ戻る
- [ ] 戦略俯瞰の距離やPitchが標準側へ残らない
- [ ] F6往復でPLAYER位置が変わらない
- [ ] F6往復で目的地／NavPathが消えない
- [ ] F6往復でWALK／RUN／DASH状態が変わらない
- [ ] F6往復で選択中NPCが解除されない

## 6. Camera CollisionとUI

- [ ] 壁・柱・建物へ近づいてもカメラが極端にめり込まない
- [ ] 障害物から離れるとカメラ距離が自然に戻る
- [ ] Collision復帰時に激しい跳ね・長い停止がない
- [ ] 通常UI上の右クリックがカメラ操作へ抜けない
- [ ] 通常UI上のホイールがWorldカメラへ抜けない
- [ ] F1技術表示に現在のカメラモード、Yaw、Pitch、距離、FOVが表示される
