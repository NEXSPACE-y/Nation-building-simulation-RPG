# Stage D 操作方法

## 基本操作

| 操作 | キー |
|---|---|
| 前後左右へ移動 | `W` / `S` / `A` / `D` |
| カメラ回転 | マウス移動 |
| 350uu以内にいるNPCを順番に選択 | `Tab` |
| 会話 | `1`（TALK） |
| 支援 | `2`（HELP） |
| 危害 | `3`（HARM） |
| 取引 | `4`（TRADE） |
| 盗む | `5`（STEAL） |
| 待機 | `6`（WAIT） |
| 次の場所へ移動 | `7`（MOVE） |
| デバッグ表示切替 | `F1` |
| 連鎖途中保存 | `F5` |
| 保存読込 | `F9` |

## NPCを選択して行動する手順

1. NPCの頭上にある`npc_id`と役割表示を確認する。
2. 対象NPCから350 Unreal Units以内までWASDで近づく。
3. `Tab`を押す。範囲内に複数のNPCがいる場合は、目的のNPCになるまで繰り返し押す。
4. 選択されたNPCの頭上に、ピンク色の`[ TARGET ]`マーカーが表示されることを確認する。
5. HUDの`TARGET:`欄に表示された`npc_id`を確認する。
6. `1`～`5`の行動キーを押す。

`Tab`で手動選択したTargetは、カメラを別方向へ向けても解除されない。NPCが350uuの範囲外へ移動した場合、別の場所へ移動した場合、またはNPCが無効になった場合だけ解除される。

## `Tab`を押しても選択されない場合

- 350uu以内に有効なNPCがいない。
- プレイヤーとNPCのロケーションIDが異なる。
- NPCが非アクティブ、破棄中、または表示ワールドに存在しない。

この場合、HUDの`Action blocked:`欄または`Recent event:`欄に理由が表示される。NPCへさらに近づいてから`Tab`を押し直す。

## 必須シナリオでの対象

1. `7`を押して`market`へ移動する。
2. `non_ai_npc_002`の表示を探し、350uu以内まで近づく。
3. `Tab`で`non_ai_npc_002`を選択する。
4. 頭上の`[ TARGET ]`とHUDの`TARGET: non_ai_npc_002`を確認する。
5. `3`を押してHARMを実行する。
